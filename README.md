# AWS Serverless Web App Practice

Power of Math

## Project Overview
This project is a simple serverless web applocation built on AWS.
The application calculate the power of a numberand stores the result in Amazon DynamoDB.

## Aechitecture

## Came up questions

### Why use DynamoDB instead of Amazon S3?
S3 is designed to store files such as images, videos, and documents.
DynamoDB is designed to store structured data and allows fast retrieval of records.
For this project, DynamoDB was a better choice because I wanted to store and retrieve calculation results.

## Challenges
Understanding IAM permissions was challenging at first.
I learned that Lambda does not automatically have access to DynamoDB. I needed to create an IAM policy and attach it to the Lambda execution role.
I also copied the DynamoDB table ARN into the policy to specify which table the Lambda function was allowed to access.

### Solution
I configured the IAM policy with the required DynamoDB permissions and attached it to the Lambda execution role.
After that, the Lambda function was able to access the DynamoDB table securely.
